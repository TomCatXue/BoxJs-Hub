# BoxJs Hub

聚合一键订阅中心 - BoxJs 模块商店

## 项目简介

这是一个基于 Vue 3 + Tailwind CSS 构建的 BoxJs 模块聚合订阅页面，支持：
- 一键订阅各大作者的 BoxJs 模块
- 分类浏览和标签筛选
- 支持 Relay/BoxJs 地址自定义

## 快速开始

### 本地开发

```bash
# 直接用浏览器打开 index.html 即可
```

### 部署到 GitHub Pages

1. 创建一个 GitHub 仓库（建议命名为 `boxjs-hub` 或自定义）
2. 将本项目所有文件推送到仓库
3. 进入仓库 Settings → Pages → 选择 `main` 分支 → Save
4. 等待几分钟后，访问 `https://<你的用户名>.github.io/<仓库名>/`

## 文件说明

- `index.html` - 主页面文件
- `store.json` - 模块数据文件
